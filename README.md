# Learn Git

## Basic git interaction
  * Do changes
  * Add files to staging area
  * Commit changes


## Git commands cheat sheet

### - Init empty git repository
```
$ git init
```

### - Add files to Staging Area
```
$ git add [filename]
```
to add all files
```
$ git add .
```

### - Check the changes in staging area
```
$ git status
```
